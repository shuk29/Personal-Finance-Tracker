# Personal Finance Tracker

A command-line Personal Finance Tracker built with Python.

## Features

- Add income and expense transactions
- Save transactions to a CSV file
- View transactions within a date range
- Display total income, expenses, and savings
- Plot income and expense trends using Matplotlib

## Technologies

- Python
- Pandas
- Matplotlib
- CSV

## Project Structure

- `main.py` – Main application
- `data_entry.py` – User input validation
- `finance_data.csv` – Transaction data

## How to Run

1. Install the required packages:

```bash
pip install pandas matplotlib
```

2. Run:

```bash
python3 main.py
```
